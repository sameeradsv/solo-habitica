# Solo Habitica

A solo, local-first habit tracker inspired by Habitica — featuring RPG-style XP, gold, difficulty levels, and progress tracking. No parties, no collaboration, just you vs your habits.

## Features

- **Local-first**: All data stored in your browser's localStorage - no server needed
- **RPG Elements**: Earn XP and gold by completing tasks
- **Difficulty Levels**: 
  - Easy: +10 XP, +5 Gold
  - Medium: +20 XP, +10 Gold  
  - Hard: +40 XP, +20 Gold
- **Level System**: Level up every 100 XP
- **Simple UI**: Clean, dark-themed interface
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript

## How to Use

1. Open `index.html` in your browser
2. Add tasks with your preferred difficulty level
3. Complete tasks to earn XP and gold
4. Level up as you make progress!

No installation required! Just:
1. Clone this repository
2. Open `index.html` in your browser
3. Start tracking your habits

## Privacy

All your data stays in your browser. Nothing is sent to any server.
